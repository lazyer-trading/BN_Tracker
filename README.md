# 追客（tracker）币安公域跟单信号追踪
![alt text](image-7.png)

追客软件将实时跟踪币安公域跟单的历史交易进行仓位信号整理，支持【图形界面/命令行】运行，支持windows/linux系统，支持实时仓位变动信息telegram转发。目前追客只支持交易信号获取，实时同步跟单交易 魅客(MAKER)还在开发中。
